# Simple Twitter Client

## Installation

1. Clone this repo
2. `composer install`
3. `cp .env.example .env`
3. `npm install`
4. `npm run dev`
5. `php artisan key:generate`
6. Setup your Twitter application and fill the `TWITTER_*` attributes in `.env`
7. `php artisan serve`
8. Open browser: `http://localhost:8000`
